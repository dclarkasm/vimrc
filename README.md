# vimrc
My vimrc...

## Quickstart
To create a symlink at `~/.vimrc` to this vimrc, run:
`> $ ./setup.sh`

Note - this script also moves the old `~/.vimrc` to `~/.vimrc.old`.

## Details
Create a symlink to this vimrc:

`> $ ln -s vimrc ~/.vimrc`

or simply copy it to `~/.vimrc`:
`> $ cp vimrc ~/.vimrc`
